NAVIGATION

INFO (F11) show help
DISPLAY (INS) change theme layout
MENU (F10) jumps to now playing screen
CHANNELUP (PgUP) previous fanart image
CHANNELDN (PgDOWN) next fanart image
BACK, EXIT (BACKSPACE, ESC) show shutdown screen

PLAYER CONTROLS

PLAY (SPACE) play song
PAUSE (PAUSE) pause song
STOP (NUM5) stop playing
PREV (NUM8) previous song
NEXT (NUM2) next song
FFWD (NUM6) forward
REW (NUM4) rewind
